コメント


　まとんは C++ と同じコメントを持つ。
　また、加えて /-- ... --/ という区切り型のコメントも有効である。

Examples:

// これは通常のライン・コメント

/*　複数行コメントも有効。
　　これでいろいろとサマリーが書けます。
 */

/-- シフト・キーを押すのが面倒なので、
　　このコメント形式も有効です。
 --/

コメント
:= 　"//" ( 任意文字 - 行端 )
|　　"/--" ( 任意文字 - "--/" )* "--/"
|　　"/*" ( 任意文字 - "*/" )* "*/"
;
